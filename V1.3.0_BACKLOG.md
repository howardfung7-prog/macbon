# MacBon v1.3.0 Backlog

Changes accumulated since v1.2.0. Ship as a single signed/notarized release when this list grows to 3-5 items.

## Done — waiting to ship

- [x] **Hardware UUID rebind protection** (`MiningTracker.resolveDeviceID`)
  Wipes local state if the cached `device_id` doesn't match the current Mac's hardware UUID. Prevents a copied `UserDefaults` plist from making the UI display another Mac's balance.

## Release procedure

When ready to ship:

```bash
# 1. Bump version
# Edit TapMac/Info.plist CFBundleShortVersionString → 1.3.0
# Edit MacBon.xcodeproj/project.pbxproj MARKETING_VERSION → 1.3.0

# 2. Update RELEASE_NOTES.md with shipped items from this file

# 3. Build (on the Mac with Developer ID cert)
cd ~/Desktop/Claude\ Code/macbon
./release.sh

# 4. Commit + tag + release
git add -A && git commit -m "v1.3.0: ..."
git tag v1.3.0 && git push --tags
gh release create v1.3.0 dist/MacBon.dmg \
  --title "MacBon 1.3.0" \
  --notes-file RELEASE_NOTES.md

# 5. Update website download link
sed -i '' 's|releases/download/v1.2.0/MacBon.dmg|releases/download/v1.3.0/MacBon.dmg|g' website/index.html
cd website && vercel --prod
```
